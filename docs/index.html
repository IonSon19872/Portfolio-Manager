🚀 How to Deploy — GitHub Actions (100% Free, Always On)

1
Create a GitHub repository (private recommended)
Go to github.com/new → create a new private repo.
2
Upload the project files to the repo root
Drag and drop the unzipped folder contents, or use git: 
git init && git remote add origin https://github.com/YOUR_USER/YOUR_REPO.git
git add . && git commit -m "Portfolio monitor"
git push -u origin main
3
Edit your portfolio — open portfolio_config.json in the repo and replace the sample stocks with your own. Use Yahoo Finance tickers for display; Finnhub symbols auto-converted. E.g. ASML.AS · SAP.DE · AAPL etc.
4
Add GitHub Secrets — Settings → Secrets and variables → Actions:
EMAIL_FROMyour.gmail@gmail.com
EMAIL_PASSWORDGmail App Password — Google Account → Security → 2FA → App Passwords
EMAIL_TOWhere to receive alerts
FINNHUB_API_KEYFree at finnhub.io — 60 req/min (register in 30 s)
5
Enable GitHub Pages for the dashboard:
Repo Settings → Pages → Source: main branch, /docs folder → Save.
Your dashboard will be at: https://YOUR_USER.github.io/YOUR_REPO/
6
GitHub Actions runs automatically:
• Price digest: 07:45 · 12:00 · 15:15 · 17:00 CET (Mon–Fri)
• Intelligence (ratings + news): 07:30 · 17:00 CET (Mon–Fri)
• Trigger manually anytime: repo → Actions tab → pick workflow → Run workflow

After each run, updated JSON is committed back to the repo. Reload this page to see the latest data.
7
Summer/Winter time note: GitHub Actions uses UTC. The workflow files use CET (UTC+1). In summer (CEST, UTC+2) you may want to adjust the cron times by 1 hour. Open .github/workflows/price_digest.yml and update the cron expressions.
